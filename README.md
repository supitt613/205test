# 西瓜倒數提醒 (Watermelon Focus Timer)這是一個23分鐘的西瓜倒數提醒應用，旨在幫助使用者專注學習或工作，並在時間結束時透過 Line Notify 發送提醒。同時，每次計時結束也會將會話記錄到 Supabase 資料庫中。## 技術棧- **前端**: HTML, CSS, JavaScript- **後端**: Node.js (Express)- **通知**: Line Notify API- **資料庫**: Supabase## 功能- 顯示23分鐘倒數計時器- 開始、暫停、重設計時器功能- 計時結束時播放提示音和瀏覽器彈窗提醒- 計時結束時透過 Line Notify 發送通知訊息- 每次計時結束後，將會話資訊（開始時間、持續時間、是否已通知）記錄到 Supabase 資料庫## 檔案結構```.├── public/│   ├── index.html        # 前端頁面│   ├── style.css         # 前端樣式│   └── script.js         # 前端邏輯 (計時器控制, 後端 API 呼叫)├── server.js             # 後端伺服器 (Express, Line Notify, Supabase 整合)├── package.json          # Node.js 專案依賴管理├── .env                  # 環境變數設定 (API 金鑰等)└── README.md             # 專案說明文件```## 設定與運行### 1. 克隆專案```bashgit clone <你的專案儲存庫網址>cd <你的專案目錄>```### 2. 環境變數設定在專案根目錄下創建一個 `.env` 檔案，並填入以下資訊：```dotenvPORT=3000LINE_NOTIFY_TOKEN=pqQJK9wqCQjonHOS11e2p/sPgOyOSCc4D5QblIuqPXHcuMyPbJDYSu1EFdUzpnVwtbgNojZozakEWm/Z3zX1HjYqlXKOW1RCE0mblyw88IUgHVn/BIQS08IKc5RsvdnwtSZ+oCaOxqyuLYF6q9BIVdGB04t89/1O/w1cDnyilFU=SUPABASE_URL=https://lerdhfducrmetbrletyt.supabase.coSUPABASE_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6ImxlcmRoZmR1Y3JtZXRicmxldHl0Iiwicm9sZSI6InNlcnZpY2Vfcm9sZSIsImlhdCI6MTc2ODc5OTUxNiwiZXhwIjoyMDg0Mzc1NTE2fQ.uhkjNAQvYelbnebzOR_pBYAKrop0HTgUgltZF0kw7EI```- `PORT`: 後端伺服器運行的埠號。- `LINE_NOTIFY_TOKEN`: 你的 Line Notify 存取權杖。請確保這個權杖已與你希望接收通知的聊天室綁定。- `SUPABASE_URL`: 你的 Supabase 專案 URL。- `SUPABASE_KEY`: 你的 Supabase `service_role` 金鑰（因為是在後端使用）。### 3.
